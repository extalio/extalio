Going through Twitter and blogs every day and filtering through the junk takes too much time. We figured others are having the same problem. Which is why we decided to compile a list of the interesting content that was published lately. We will publish every few weeks (when enough exciting things have happened).

Vulnerabilities

Privilege Escalation in Ubuntu snapd daemon

Snappy is a package manager developed by Canonical that is installed on Ubuntu by default. The snapd daemon exposes an HTTP server listening on two UNIX sockets with 0666 permissions. The HTTP server allows some high privileged operations after checking the uid of the socket’s source. The uid checking can be abused through the socket file name which gets concatenated to the same string and then split. This can allow an attacker to execute high privileged operations such as user creation, leading to local privilege escalation.

CVE-2019-5736: Escape from Docker and Kubernetes containers to root on host

A vulnerability in runc (CLI tool for running containers used by Docker, Kubernetes, etc.) that allows a malicious container to escape to host. An attacker can trick runc to execute itself inside the container which then gains permissions to change the runc binary on the host. The article discusses their approaches during the research, the failed attempts, and the fixes.

The Curious Case of Convexity Confusion

A vulnerability found in Skia (a graphics library used by Chrome, Firefox, Android, and more) after an insufficient fix of another vulnerability (CVE-2018-6153). Tiny floating point values cause precision issues which affect a shape’s attribute during type conversion. This attribute may cause a wrong drawing algorithm execution, resulting in a negative value when it shouldn’t be possible which leads to an out-of-bounds read/write on the stack.

Reverse RDP Attack: Code Execution on RDP Clients #madeinisrael

Researching RDP clients using a malicious RDP server. Explains and demonstrates the found vulnerabilities: RCE in FreeRDP and rdesktop due to heap overflows caused by incorrect memory allocations after an integer underflow. Also found a path traversal vulnerability in mstsc when copying files.

Interesting Research

Examining Pointer Authentication on the iPhone XS

Pointer Authentication (PAC) is a new feature of ARM processors. The article presents several weaknesses with the PAC implementation on iPhone XS. It describes the strategy for forging the key from which the PACs are created.

Make It Rain with MikroTik

A step by step guide for installing Mikrotik RouterOS on a VM and getting root privileges, finding the binaries on the system, and communicating with them.

A Brief History of BaseNamedObjects on Windows NT

A full overview of the poorly documented BaseNamedObjects mechanism in windows (used for instance in CreateEvent function). James gives a full historical overview of the feature itself and discusses the changes made in Windows 10 which for the first time in history make this feature suitable for process isolation.

Tools

ClusterFuzz

Scalable fuzzing infrastructure used by Google for fuzzing with OSS-Fuzz. Among the included features: automatic bug filing, test case minimization, regression finding, statistics for analyzing fuzzer performance, and an easy to use web interface.

CANalyzat0r

A tool for CAN-bus (a central bus in vehicles) analysis that includes logging and packet sniffing capabilities under a friendly GUI interface.

mesos

A tool for binary code coverage, applying breakpoints on every single block in a program and removing them as they are hit. This is essentially a very fast debugger. Automatically collects minidump of a crashed process that runs under it. Can be modified to create breakpoints handlers to inject inputs (and build fuzzer on top of it).
